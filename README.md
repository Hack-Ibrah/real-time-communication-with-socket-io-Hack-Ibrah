# Week5 Real-Time Chat (Socket.io) - Ready Submission

This package contains a simple real-time chat app with Socket.io.

Folders:
- server : Express + Socket.io server (in-memory demo store)
- client : Vite + React client

Advanced features included (>=3):
- Private messaging support
- Typing indicators
- Online/offline user list
- Message reactions
- Read receipts
- Notifications (basic sound + socket notifications)

How to run locally:
1. Start server:
   cd server
   npm install
   npm run dev

2. Start client:
   cd ../client
   npm install
   VITE_SERVER_URL=http://localhost:4000 npm run dev
