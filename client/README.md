# Week5 Socket.io Client (Vite + React)

Run:
1. cd client
2. npm install
3. VITE_SERVER_URL=http://localhost:4000 npm run dev

Features implemented:
- Global chat room
- Simple username "login" that returns JWT token
- Typing indicator
- Online users list
- Private messaging (server supports it) - can be extended in UI
- Message reactions and read marking
